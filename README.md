# Arduino_CAN-project
Making two Arduinos communicate through CAN Bus.
## Major Components 
### Transmitter
- Arduino UNO Board
- DHT11 Sensor
- HC-SR04 Ultrasonic Sensor
- IR Proximity Sensor Module
- CAN Module MCP2515
### Receiver
- Arduino UNO Board
- I2C LCD Display
- CAN Module MCP2515
